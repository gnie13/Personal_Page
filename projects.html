Spotify API Music Tracker

This project uses the Spotify API to display real-time information on the music users are currently listening to. The website is styled with custom CSS, including a gradient background, unique button and card designs, and interactive hover effects. The application connects with Spotify's API to pull details on the currently played song, album art, and artist information, offering an immersive music tracking experience.

Key Features:

  • Dynamic integration with Spotify's API to fetch real-time music data.
  • Custom-designed cards that display album art and track information.
  • Interactive user interface with gradient backgrounds and hover animations for buttons and cards.
  • JavaScript functionality managed through the album.js file, with HTML structure in album.html.
